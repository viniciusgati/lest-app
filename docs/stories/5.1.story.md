# Story 5.1 — Backend — Métricas de Desempenho

## Status: Approved

## Story

**Como** usuário do EduTrack,
**Quero** que o backend calcule meu desempenho por matéria e tema,
**Para que** eu possa visualizar onde estou indo bem e onde preciso melhorar.

---

## Contexto

Primeira story do Epic 5. Sessões de estudo existem (Epic 4). Esta story cria endpoints que agregam os dados das sessões completadas para calcular métricas de desempenho.

Não há tabela de métricas separada — tudo é calculado sob demanda a partir das `study_sessions` completadas. O endpoint também retorna o progresso real da meta semanal atual.

**Dependência:** Epics 1–4 completos.

---

## Acceptance Criteria

### Métricas por matéria
- [ ] **AC1** — `GET /api/v1/metrics/subjects` retorna lista de matérias com `accuracy_percentage` calculado
- [ ] **AC2** — `accuracy_percentage` = `SUM(questions_correct) / SUM(questions_done) * 100` das sessões completadas da matéria
- [ ] **AC3** — Matérias sem sessões completadas retornam `accuracy_percentage: null`

### Métricas por tema
- [ ] **AC4** — `GET /api/v1/metrics/subjects/:subject_id/topics` retorna temas com `accuracy_percentage` por tema
- [ ] **AC5** — Inclui histórico de sessões do tema (últimas 10, ordenadas por data desc)

### Progresso semanal
- [ ] **AC6** — `GET /api/v1/metrics/weekly_progress` retorna progresso real da semana atual vs meta
- [ ] **AC7** — `actual_hours` = soma de `actual_minutes / 60` das sessões completadas na semana
- [ ] **AC8** — `actual_questions` = soma de `questions_done` das sessões completadas na semana
- [ ] **AC9** — `actual_percentage` = média ponderada de acertos das sessões completadas na semana
- [ ] **AC10** — Retorna também a meta atual (`target_*`) para o frontend exibir progresso

### Geral
- [ ] **AC11** — Todos os endpoints retornam 401 sem JWT
- [ ] **AC12** — RSpec cobre todos os endpoints com dados reais de sessões

## Dev Notes

### MetricsController

```ruby
# app/controllers/api/v1/metrics_controller.rb
module Api
  module V1
    class MetricsController < BaseController
      def subjects
        subjects = current_user.subjects.includes(:topics)

        result = subjects.map do |subject|
          sessions = completed_sessions_for_subject(subject)
          {
            id: subject.id,
            name: subject.name,
            accuracy_percentage: calculate_accuracy(sessions),
            sessions_count: sessions.count
          }
        end

        render json: result
      end

      def subject_topics
        subject = current_user.subjects.find(params[:subject_id])
        topics = subject.topics.includes(:study_sessions)

        result = topics.map do |topic|
          sessions = topic.study_sessions.where(status: 'completed').order(scheduled_date: :desc)
          {
            id: topic.id,
            name: topic.name,
            notes: topic.notes,
            next_review: topic.next_review,
            ease_factor: topic.ease_factor,
            interval: topic.interval,
            accuracy_percentage: calculate_accuracy(sessions),
            sessions_count: sessions.count,
            recent_sessions: sessions.limit(10).map { |s| session_summary(s) }
          }
        end

        render json: result
      rescue ActiveRecord::RecordNotFound
        render json: { error: 'Not found' }, status: :not_found
      end

      def weekly_progress
        week_start = Date.today.beginning_of_week(:monday)
        week_end   = week_start + 6.days

        sessions = completed_sessions_in_range(week_start, week_end)
        goal     = current_user.weekly_goals.find_by(week_start: week_start)

        actual_hours      = sessions.sum('actual_minutes').to_f / 60
        actual_questions  = sessions.sum(:questions_done)
        actual_percentage = calculate_accuracy(sessions)

        render json: {
          week_start: week_start,
          actual_hours: actual_hours.round(1),
          actual_questions: actual_questions,
          actual_percentage: actual_percentage,
          target_hours: goal&.target_hours || 0,
          target_questions: goal&.target_questions || 0,
          target_percentage: goal&.target_percentage || 0
        }
      end

      private

      def completed_sessions_for_subject(subject)
        StudySession.joins(topic: :subject)
                    .where(subjects: { id: subject.id }, status: 'completed')
      end

      def completed_sessions_in_range(from, to)
        StudySession.joins(topic: :subject)
                    .where(subjects: { user_id: current_user.id },
                           status: 'completed',
                           scheduled_date: from..to)
      end

      def calculate_accuracy(sessions)
        total_done    = sessions.sum(:questions_done)
        total_correct = sessions.sum(:questions_correct)
        return nil if total_done.zero?
        (total_correct.to_f / total_done * 100).round(1)
      end

      def session_summary(session)
        {
          id: session.id,
          scheduled_date: session.scheduled_date,
          questions_done: session.questions_done,
          questions_correct: session.questions_correct,
          accuracy: session.questions_done > 0
                    ? (session.questions_correct.to_f / session.questions_done * 100).round(1)
                    : nil
        }
      end
    end
  end
end
```
