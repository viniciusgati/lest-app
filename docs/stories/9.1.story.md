# Story 9.1 — Frontend — Timer de Sessão In-app

## Status: Ready for Review

## Story

**Como** usuário do EduTrack,
**Quero** cronometrar minhas sessões de estudo diretamente no app,
**Para que** o campo `actual_minutes` seja preenchido automaticamente ao registrar o resultado.

---

## Contexto

Atualmente o campo `actual_minutes` raramente é preenchido porque o usuário precisa saber quanto tempo estudou. A maioria das sessões tem `actual_minutes: null` nos dados. Esta story implementa um componente de timer (Start / Pause / Stop) na tela de Agenda, dentro do card/modal de sessão ativa, que ao ser parado preenche `actual_minutes` automaticamente no formulário de conclusão.

**Fluxo:**
1. Usuário abre sessão agendada na Agenda
2. Clica "Iniciar Timer" → cronômetro começa (HH:MM:SS)
3. Pode pausar/retomar
4. Ao clicar "Parar" → tempo total é convertido para minutos e preenche o campo no formulário de conclusão
5. Usuário confirma resultado → session marcada como `completed` com `actual_minutes` correto

## Dev Notes

### Estrutura do componente

```typescript
// frontend/src/app/shared/components/study-timer/study-timer.component.ts
@Component({
  selector: 'app-study-timer',
  standalone: true,
  imports: [CommonModule, ButtonModule],
  template: `...`
})
export class StudyTimerComponent implements OnDestroy {
  status: 'idle' | 'running' | 'paused' = 'idle';
  elapsedMs = 0;
  private intervalId?: ReturnType<typeof setInterval>;

  @Output() timerStopped = new EventEmitter<number>();

  start() {
    this.status = 'running';
    this.intervalId = setInterval(() => this.elapsedMs += 1000, 1000);
  }

  pause() {
    this.status = 'paused';
    clearInterval(this.intervalId);
  }

  resume() {
    this.start();
  }

  stop() {
    clearInterval(this.intervalId);
    this.status = 'idle';
    const minutes = Math.ceil(this.elapsedMs / 60000);
    this.timerStopped.emit(minutes);
    this.elapsedMs = 0;
  }

  get displayTime(): string {
    const totalSec = Math.floor(this.elapsedMs / 1000);
    const h = Math.floor(totalSec / 3600);
    const m = Math.floor((totalSec % 3600) / 60);
    const s = totalSec % 60;
    if (h > 0) return `${h}:${m.toString().padStart(2,'0')}:${s.toString().padStart(2,'0')}`;
    return `${m.toString().padStart(2,'0')}:${s.toString().padStart(2,'0')}`;
  }

  ngOnDestroy() {
    clearInterval(this.intervalId);
  }
}
```

### Integração no formulário de conclusão

```typescript
// No componente pai (agenda ou modal de sessão)
onTimerStopped(minutes: number) {
  const ctrl = this.completionForm.get('actual_minutes');
  if (!ctrl?.value) {  // não sobrescreve se já preenchido
    ctrl?.setValue(minutes);
  }
}
```
