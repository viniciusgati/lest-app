# Story 4.2 — Frontend — Calendário + Agendamento de Sessões

## Status: Draft

## Story

**Como** usuário do EduTrack,
**Quero** visualizar minhas sessões de estudo num calendário e agendar novas revisões,
**Para que** eu tenha visibilidade do meu plano de estudos e possa organizá-lo manualmente.

---

## Contexto

Continuação da Story 4.1. Backend de sessões pronto. Esta story implementa a tela `/agenda` com calendário mensal/semanal usando PrimeNG `p-fullCalendar` ou `p-datePicker` e um form de agendamento de sessões.

**Dependência:** Stories 4.1 concluída.

export interface StudySession {

### Mapeamento de cores por status

```typescript
const statusColors: Record<string, string> = {
  scheduled: '#3b82f6', // azul
  completed: '#22c55e', // verde
  late: '#ef4444'       // vermelho
};

// Mapear sessão para evento FullCalendar
function sessionToEvent(session: StudySession, topicName: string): EventInput {
  return {
    id: String(session.id),
    title: topicName,
    date: session.scheduled_date,
    backgroundColor: statusColors[session.status],
    borderColor: statusColors[session.status],
    extendedProps: { session }
  };
}
```

### StudySessionService

```typescript
@Injectable({ providedIn: 'root' })
export class StudySessionService {
  private readonly url = '/api/v1/study_sessions';

  constructor(private http: HttpClient) {}

  getAll(filters?: { date_from?: string; date_to?: string }): Observable<StudySession[]> {
    return this.http.get<StudySession[]>(this.url, { params: filters ?? {} });
  }

  create(session: Partial<StudySession>): Observable<StudySession> {
    return this.http.post<StudySession>(this.url, { study_session: session });
  }

  update(id: number, session: Partial<StudySession>): Observable<StudySession> {
    return this.http.put<StudySession>(`${this.url}/${id}`, { study_session: session });
  }

  delete(id: number): Observable<void> {
    return this.http.delete<void>(`${this.url}/${id}`);
  }

  complete(id: number, result: StudySessionResult): Observable<{ session: StudySession; next_session: StudySession }> {
    return this.http.put<any>(`${this.url}/${id}/complete`, { study_session: result });
  }
}
```
