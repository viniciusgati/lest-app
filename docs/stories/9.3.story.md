# Story 9.3 — Backend + Frontend — Métricas Históricas (8 semanas)

## Status: Draft

## Story

**Como** usuário do EduTrack,
**Quero** visualizar minha evolução de desempenho nas últimas 8 semanas,
**Para que** eu possa identificar tendências de melhora ou piora por matéria ao longo do tempo.

---

## Contexto

As métricas atuais são snapshots: mostram accuracy total acumulado. Não há visão temporal. Esta story adiciona um endpoint de histórico semanal e um gráfico de linha no dashboard mostrando accuracy por semana para cada matéria.

**Dado disponível:** `study_sessions` tem `scheduled_date` e `status: 'completed'` com `questions_correct` e `questions_done`. Podemos agrupar por semana (início da semana = segunda-feira) e calcular accuracy semanal.

**Escopo:** últimas 8 semanas com dados (semanas sem sessões completadas aparecem como ponto vazio ou são ignoradas).

---

## Acceptance Criteria

- [ ] **AC1** — `GET /api/v1/metrics/history` retorna dados de accuracy por semana das últimas 8 semanas
- [ ] **AC2** — Formato da resposta: `{ subjects: [{ id, name, weeks: [{ week_start, accuracy, sessions_count }] }] }`
- [ ] **AC3** — `week_start` é sempre segunda-feira no formato ISO 8601 (`YYYY-MM-DD`)
- [ ] **AC4** — Semanas sem sessões completadas são omitidas (não retornam ponto com accuracy null)
- [ ] **AC5** — `accuracy` é calculado como `SUM(questions_correct) / SUM(questions_done)` na semana (0.0 a 1.0)
- [ ] **AC6** — Parâmetro opcional `?weeks=N` (default: 8, max: 26)
- [ ] **AC7** — Frontend: gráfico de linha na tela Home/Dashboard exibindo accuracy semanal por matéria
- [ ] **AC8** — Gráfico usa PrimeNG `Chart` (tipo `line`) com uma série por matéria
- [ ] **AC9** — Gráfico mostra apenas matérias com pelo menos 2 semanas de dados (para ter linha)
- [ ] **AC10** — RSpec cobre: cálculo por semana, múltiplas matérias, parâmetro `weeks`

## Dev Notes

### Query SQL (backend)

```ruby
# MetricsController#history
def history
  weeks = [(params[:weeks] || 8).to_i, 26].min
  since_date = weeks.weeks.ago.beginning_of_week(:monday)

  results = StudySession
    .joins(topic: :subject)
    .where(subjects: { user_id: current_user.id })
    .where(status: 'completed')
    .where('study_sessions.scheduled_date >= ?', since_date)
    .where('study_sessions.questions_done > 0')
    .group(
      "DATE_TRUNC('week', study_sessions.scheduled_date + INTERVAL '1 day') - INTERVAL '1 day'",
      'subjects.id',
      'subjects.name'
    )
    .select(
      "DATE_TRUNC('week', study_sessions.scheduled_date + INTERVAL '1 day') - INTERVAL '1 day' AS week_start",
      'subjects.id AS subject_id',
      'subjects.name AS subject_name',
      'SUM(study_sessions.questions_correct)::float / SUM(study_sessions.questions_done) AS accuracy',
      'COUNT(*) AS sessions_count'
    )

  # Organizar por subject
  by_subject = results.group_by(&:subject_id)
  render json: {
    subjects: by_subject.map do |_id, rows|
      {
        id: rows.first.subject_id,
        name: rows.first.subject_name,
        weeks: rows.map { |r|
          {
            week_start: r.week_start.to_date.iso8601,
            accuracy: r.accuracy.round(4),
            sessions_count: r.sessions_count
          }
        }.sort_by { |w| w[:week_start] }
      }
    end
  }
end
```

### Interfaces TypeScript

```typescript
interface WeekData {
  weekStart: string;  // 'YYYY-MM-DD'
  accuracy: number;   // 0.0 a 1.0
  sessionsCount: number;
}

interface SubjectHistory {
  id: number;
  name: string;
  weeks: WeekData[];
}

interface MetricsHistory {
  subjects: SubjectHistory[];
}
```

### PrimeNG Chart (Chart.js)

```typescript
// Transformar dados para Chart.js
buildChartData(history: MetricsHistory): ChartData {
  // Coletar todas as semanas únicas
  const allWeeks = [...new Set(
    history.subjects.flatMap(s => s.weeks.map(w => w.weekStart))
  )].sort();

  const colors = ['#6366f1', '#ec4899', '#14b8a6', '#f59e0b', '#8b5cf6'];

  return {
    labels: allWeeks.map(w => new Date(w).toLocaleDateString('pt-BR', { day: '2-digit', month: '2-digit' })),
    datasets: history.subjects
      .filter(s => s.weeks.length >= 2)
      .map((subject, i) => ({
        label: subject.name,
        data: allWeeks.map(week => {
          const found = subject.weeks.find(w => w.weekStart === week);
          return found ? Math.round(found.accuracy * 100) : null;
        }),
        borderColor: colors[i % colors.length],
        tension: 0.3,
        spanGaps: true
      }))
  };
}
```
