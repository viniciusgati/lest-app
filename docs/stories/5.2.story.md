# Story 5.2 — Frontend — Dashboard Home + Métricas

## Status: Approved

## Story

**Como** usuário do EduTrack,
**Quero** ver na Home meu desempenho por matéria e o progresso da minha meta semanal,
**Para que** eu tenha uma visão geral rápida do meu estudo ao abrir o app.

---

## Contexto

Continuação da Story 5.1. Endpoints de métricas prontos. Esta story implementa a Home (`/`) com o dashboard completo: progresso da meta semanal, desempenho por matéria (com drill-down para temas), e calendário semanal com sessões.

A Home agora também exibe o mini-calendário da semana atual com as sessões de hoje e dos próximos dias.

**Dependência:** Stories 4.2 e 5.1 concluídas.

### MetricsService

```typescript
@Injectable({ providedIn: 'root' })
export class MetricsService {
  private readonly base = '/api/v1/metrics';

  constructor(private http: HttpClient) {}

  getSubjectsMetrics(): Observable<SubjectMetric[]> {
    return this.http.get<SubjectMetric[]>(`${this.base}/subjects`);
  }

  getWeeklyProgress(): Observable<WeeklyProgress> {
    return this.http.get<WeeklyProgress>(`${this.base}/weekly_progress`);
  }
}
```

### HomeComponent

```typescript
@Component({ selector: 'app-home', standalone: true, ... })
export class HomeComponent implements OnInit {
  weeklyProgress: WeeklyProgress | null = null;
  subjectsMetrics: SubjectMetric[] = [];
  weekSessions: StudySession[] = [];
  loading = false;

  readonly weekDays = ['Seg','Ter','Qua','Qui','Sex','Sáb','Dom'];

  ngOnInit() {
    this.loading = true;
    const today = new Date();
    const weekStart = this.getMonday(today).toISOString().split('T')[0];
    const weekEnd   = this.getSunday(today).toISOString().split('T')[0];

    forkJoin({
      progress: this.metricsService.getWeeklyProgress(),
      subjects: this.metricsService.getSubjectsMetrics(),
      sessions: this.sessionService.getAll({ date_from: weekStart, date_to: weekEnd })
    }).subscribe({
      next: ({ progress, subjects, sessions }) => {
        this.weeklyProgress = progress;
        this.subjectsMetrics = subjects;
        this.weekSessions = sessions;
        this.loading = false;
      },
      error: () => { this.loading = false; }
    });
  }

  sessionsForDay(date: string): StudySession[] {
    return this.weekSessions.filter(s => s.scheduled_date === date);
  }

  isToday(date: string): boolean {
    return date === new Date().toISOString().split('T')[0];
  }

  private getMonday(d: Date): Date {
    const day = d.getDay(), diff = d.getDate() - day + (day === 0 ? -6 : 1);
    return new Date(d.setDate(diff));
  }

  private getSunday(d: Date): Date {
    const monday = this.getMonday(new Date(d));
    return new Date(monday.setDate(monday.getDate() + 6));
  }
}
```
