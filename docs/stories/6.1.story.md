# Story 6.1 — Backend — Agendamento Automático da Semana

## Status: Approved

## Story

**Como** usuário do EduTrack,
**Quero** que o sistema gere automaticamente minha programação semanal de estudos,
**Para que** eu não precise agendar cada sessão manualmente e o sistema priorize o que preciso estudar.

---

## Contexto

Primeira story do Epic 6. Esta story implementa o endpoint de geração automática da semana. Usa `UserConfig` (dias disponíveis, estratégia) e `WeeklyGoal` (horas alvo) para distribuir sessões de estudo nos dias disponíveis da semana.

**Três estratégias implementadas:**
- `sm2` — prioriza temas com `next_review` vencido ou vencendo esta semana
- `weak_points` — prioriza temas com menor `accuracy_percentage`
- `balanced` — distribui igualmente entre todas as matérias

**Dependência:** Epics 1–5 completos.

---

## Acceptance Criteria

- [x] **AC1** — `POST /api/v1/schedule/generate` gera sessões para a semana atual
- [x] **AC2** — Sessões são criadas apenas para dias disponíveis em `UserConfig.available_days`
- [x] **AC3** — Minutos por dia = `(target_hours * 60) / dias_disponíveis_na_semana`
- [x] **AC4** — Estratégia `sm2`: prioriza topics com `next_review <= fim_da_semana`, depois os atrasados
- [x] **AC5** — Estratégia `weak_points`: ordena topics por `accuracy_percentage` ASC (piores primeiro), nulls primeiro

- [x] **AC7** — Sessões existentes `scheduled` na semana atual são removidas antes de gerar novas (`auto_generated: true` apenas)
- [x] **AC8** — Sessões `completed` nunca são removidas
- [x] **AC9** — Retorna lista de sessões criadas
- [x] **AC10** — Se não há `WeeklyGoal` para a semana, usa `target_hours: 2` como default por dia
- [x] **AC11** — RSpec cobre as três estratégias e a distribuição de dias

## Dev Notes

### ScheduleGeneratorService

```ruby
# app/services/schedule_generator.rb
class ScheduleGenerator
  DAYS_MAP = { 'mon' => 1, 'tue' => 2, 'wed' => 3, 'thu' => 4,
               'fri' => 5, 'sat' => 6, 'sun' => 0 }.freeze

  def self.generate(user)
    new(user).generate
  end

  def initialize(user)
    @user = user
    @config = user.user_config || UserConfig.new(
      available_days: %w[mon tue wed thu fri],
      schedule_strategy: 'sm2'
    )
    @goal = user.weekly_goals.find_by(week_start: Date.today.beginning_of_week(:monday))
    @week_start = Date.today.beginning_of_week(:monday)
    @week_end = @week_start + 6.days
  end

  def generate
    available_dates = build_available_dates
    return [] if available_dates.empty?

    minutes_per_day = calculate_minutes_per_day(available_dates.count)
    topics = select_topics

    # Remove apenas sessões auto_generated + scheduled da semana
    remove_existing_auto_sessions

    sessions = []
    topics.each_with_index do |topic, i|
      date = available_dates[i % available_dates.count]
      session = StudySession.create!(
        topic: topic,
        scheduled_date: date,
        expected_minutes: minutes_per_day,
        auto_generated: true
      )
      sessions << session
    end

    sessions
  end

  private

  def build_available_dates
    (@week_start..@week_end).select do |date|
      day_key = Date::DAYNAMES[date.wday].downcase.first(3)
      @config.available_days.include?(day_key)
    end
  end

  def calculate_minutes_per_day(days_count)
    target_hours = @goal&.target_hours || (2.0 * days_count / 7.0 * 7)
    total_minutes = (target_hours * 60).round
    (total_minutes / days_count).clamp(15, 240)
  end

  def select_topics
    case @config.schedule_strategy
    when 'sm2'        then sm2_topics
    when 'weak_points' then weak_points_topics
    else               balanced_topics
    end
  end

  def sm2_topics
    Topic.joins(subject: :user)
         .where(users: { id: @user.id })
         .where('next_review <= ?', @week_end)
         .order(:next_review)
  end

  def weak_points_topics
    # Subquery: accuracy = SUM(correct)/SUM(done) por topic
    Topic.joins(subject: :user)
         .where(users: { id: @user.id })
         .joins(<<-SQL)
           LEFT JOIN (
             SELECT topic_id,
               CASE WHEN SUM(questions_done) > 0
                 THEN SUM(questions_correct)::float / SUM(questions_done)
                 ELSE NULL END AS accuracy
             FROM study_sessions
             WHERE status = 'completed'
             GROUP BY topic_id
           ) topic_stats ON topic_stats.topic_id = topics.id
         SQL
         .order('topic_stats.accuracy ASC NULLS FIRST')
  end

  def balanced_topics
    # Round-robin: pega 1 topic de cada matéria ciclicamente
    topics_by_subject = @user.subjects.includes(:topics).flat_map do |subject|
      subject.topics.to_a
    end.group_by { |t| t.subject_id }

    result = []
    max_per_subject = topics_by_subject.values.map(&:length).max || 0
    max_per_subject.times do |i|
      topics_by_subject.each_value do |topics|
        result << topics[i] if topics[i]
      end
    end
    result
  end

  def remove_existing_auto_sessions
    StudySession.joins(topic: :subject)
                .where(subjects: { user_id: @user.id },
                       auto_generated: true,
                       status: 'scheduled',
                       scheduled_date: @week_start..@week_end)
                .destroy_all
  end
end
```

      end
    end
