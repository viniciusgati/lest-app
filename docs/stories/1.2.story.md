### Importante: API calls durante desenvolvimento

Durante o desenvolvimento local, o Angular em `ng serve` (porta 4200) precisa fazer proxy para o Rails (porta 3000). Criar `frontend/proxy.conf.json`:

```json
{
  "/api": {
    "target": "http://localhost:3000",
    "secure": false
  },
  "/health": {
    "target": "http://localhost:3000",
    "secure": false
  }
}
```

E configurar em `angular.json`:
```json
"serve": {
  "options": {
    "proxyConfig": "proxy.conf.json"
  }
}
```
